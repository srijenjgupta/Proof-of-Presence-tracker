# Proof of Presence (POP) Tracker
### A Lightweight Field Force Verification Tool
**Built by: Srijen Gupta**

---

## 📌 Overview
Proof of Presence (POP) is a B2B operational tool designed for small businesses in India to verify client visits by field staff (sales, delivery, or technicians). It uses real-time GPS geofencing to ensure staff are physically at a client's location before allowing a "Check-In."

## 🚀 Live Demo
https://srijenjgupta.github.io/Proof-of-Presence-tracker/

## 🛠️ Key Features
- **Admin Dashboard:** Register business, manage staff IDs, and assign specific clients to staff members.
- **Smart Geofencing:** 100-meter radius verification using the Haversine formula and HTML5 Geolocation.
- **Advanced Location Input:** Support for address search and direct coordinate extraction from Google Maps links.
- **Printable Territory Maps:** Generates a custom A4-ready map showing assigned client geofences.
- **Real-time Analytics:** Live attendance matrix with CSV export functionality for payroll/audit.

## 💻 Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Maps:** Leaflet.js & OpenStreetMap (100% Free)
- **Backend:** Google Firebase (Firestore Database & Authentication)
- **Deployment:** GitHub Pages

## 📖 How to Use
1. **Admin:** Register your business and save your unique Company Code.
2. **Setup:** Add staff members and assign them client locations by dropping pins on the map.
3. **Staff:** Log in with the Company Code and your unique Staff ID.
4. **Verification:** Select the client you are visiting and click "Check-In" only when you are at the site.
